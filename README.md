# twitch-multi-streamer-viewer
Twitch Multi‑Stream Viewer This page allows you to watch four Twitch streams at once and control audio independently. Each stream is shown in a 16:9 window and you can mute/unmute any of them with the on‑screen buttons.

Requirements
Visual Studio Code – download & install from code.visualstudio.com.
Live Server extension for VS Code (or any minimal HTTP server).
Install via the Extensions view (ritwickdey.LiveServer).
Start it by right‑clicking the index.html file and choosing "Open with Live Server".
